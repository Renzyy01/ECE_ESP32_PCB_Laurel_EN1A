# IDIYANALE NPK Sensor

## Project Description

The IDIYANALE NPK Sensor is a soil monitoring system designed to measure the levels of Nitrogen (N), Phosphorus (P), and Potassium (K) present in the soil. These three nutrients are essential for plant growth and play a significant role in agricultural productivity and crop health.

This project aims to provide a simple and efficient method for monitoring soil nutrient conditions. By collecting and analyzing NPK data, users can better understand soil quality and make informed decisions regarding fertilizer application and crop management.
The system is intended for educational, research, and agricultural purposes. It demonstrates the application of sensor technology, embedded systems, and data acquisition in monitoring environmental conditions. The project also serves as a learning platform for students studying electronics, programming, and agricultural technologies.

Through the integration of sensors and microcontroller-based processing, the IDIYANALE NPK Sensor provides real-time nutrient measurements that can help improve farming efficiency and promote better soil management practices.

---

## Features

- Measures Nitrogen (N) levels in soil
- Measures Phosphorus (P) levels in soil
- Measures Potassium (K) levels in soil
- Real-time soil nutrient monitoring
- Supports agricultural and educational applications
- Portable
- Rechargable
- Controlled Voltage Output
